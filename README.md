# Project-Abyss

🎯 Objetivo del Juego
Sobrevivir todo lo posible a los enemigos que aparezcan dentro del mapa, acumulando la mayor cantidad de puntaje antes de morir

🎮 Controles
Lista los controles:

Click derecho: Disparo

Mouse: Vista

W/S: Avansar, Retroceder

A/D: Strafe

Shift: Correr

Space: Abrir puertas

⚙️ Mecánicas
Movimiento fluido y audaz, cinco armas diferentes para matar a tus enemigos y un enorme mapa en el cual tendras que sobrevivir
